# Agent Module

This folder contains the LiveKit voice assistant agent for the wedding mirror.

## Files:
- `agent.py` - Main voice assistant with vision capabilities

## Usage:
```bash
cd agent
python agent.py
```

## Configuration:
Make sure to set up your environment variables in the main `.env` file:
- `LIVEKIT_URL`
- `LIVEKIT_API_KEY`
- `LIVEKIT_API_SECRET`
- `GOOGLE_AI_API_KEY`
